# MaxBlock

Минимальное Chrome-расширение для `gorzdrav.spb.ru`.

## Что делает
- Удаляет модальное окно `div.modal.agreement-modal#max-modal`.
- Работает только на `https://gorzdrav.spb.ru/*`.

## Установка (локально)
1. Откройте `chrome://extensions`.
2. Включите "Режим разработчика".
3. Нажмите "Загрузить распакованное" и выберите папку проекта.
